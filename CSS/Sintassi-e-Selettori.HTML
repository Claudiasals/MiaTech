MODULO CSS

COME SI COLLEGA UN FILE CSS?

rel specifica che si tratta di un foglio di stile, href specifica il link del file a cui collegarsi e in fine type specifica che si tratta di un file di testo di css

TIPI DI SELETTORI

  • SELETTORI SEMPLICI: selezionano gli elementi in base a nome, ID, classe.
  • SELETTORI COMBINATORI: selezionano gli elementi in base ad una relazione specifica tra loro.
  • SELETTORI DI PSEUDO-CLASSI: selezionano gli elementi in base a un determinato stato.
  • SELETTORI DI PSEUDO-ELEMENTI: selezionano una parte di un elemento.
  • SELETTORI DI ATTRIBUTO: selezionano gli elementi in base ad un attributo p valore di attributo.

SELETTORI SEMPLICI

Per NOME

        p {
            text-align: center;
            color: red;
        }

per ID

esempio html:


< p id="para1"> Hello World! < /p> 

        #para1 {
            text-align: center;
            color: red;
        }


per CLASSE

esempio html:


< p class="center"> Hello World! < /p> 

        .center {
            text-align: center;
            color: red;
        }


per una CLASSE in uno specifico ELEMENTO

esempio html:


< p class="center"> Hello World! < /p> 

        p.center {
            text-align: center;
            color: red;
        }


Selettore UNIVERSALE

Con l'* va a selezionare tutti gli elementi della pagina html 



       * {
           text-align: center;
           color: red;
       }

Selettore di GRUPPO

per selezionare più di un elemento insieme 



       p, h1, h2 {
           text-align: center;
           color: red;
       }


SELETTORI COMBINATORI

SELETTORE DISCENTENTE

seleziona gli elementi che discendono in un altro elemento: in questo caso i paragrafi contenuti nei div:


 div p {
            background-color: yellow;
        }


CHILD SELECTOR

Assegna un aregola css ai figli di un altro elemento. In questo caso a tutti i "p" figli dell'elemento "div" 
div > p {
    background-color: yellow;
}


ADJACENT SIBLING SELECTOR

L'elemento immediatamente successivo, in questo caso al primo elemento paragrafo successivo al div 


div + p {
    background-color: yellow;
}


GENERAL SIBLING SELECTOR (simbolo tilde)

div ~ p {
    background-color: yellow;
}


SELETTORI DI PSEUDO-CLASSI

si basano sullo stato di un elemento

Sintassi:

        selector:pseudo-class {
            proprerty: value;
        }


Esempio:

        div:hover {
            display: block;
        }

Questa regola significa che passando con il mouse sul div resta invariato, "bloccato" ( anche se è una caratteristica già di default dei div). Hover comunuqe è una pseudo-classe che serve per applicare uno stile quando il mouse passa sopra ad un elemento.

SELETTORI DI PSEUDO-ELEMENTI

Mentre le pseudo classi si basano sullo stato di un elemento, i selettori PSEUDO-ELEMENT ci permettono di applicare una regola css solo ad una parte del nosto elemento.

Sintassi:

        selector:pseudo-element {
            proprerty: value;
        }


Esempio:

        p::first-line {
            color: blue;
            font-variant: small-caps;
        }

Questa regola significa che la prima linea dei paragrafi sarà di colore rosso e con un font di tipo small-caps.



ATTRIBUTE-SELECTOR

Sintassi:

        selector[attribute] {
            proprerty: value;
        }


Esempio:

        a[target] {
            color: blue;
            font-variant: small-caps;
        }

In questo esempio stiamo applicando la regola dello sfondo giallo a tutti i link "a" che possiedono un attributo target.



Esempio:

        a[target="_blank"] {
            color: blue;
            font-variant: small-caps;
        }

In questo esempio invece stiamo applicando la regola dello sfondo giallo a tutti i link "a" che possiedono un attributo target che è uguale al valore "_blanc".
