TIPI DI SELETTORI

  • SELETTORI SEMPLICI: selezionano gli elementi in base a nome, ID, classe.
  • SELETTORI COMBINATORI: selezionano gli elementi in base ad una relazione specifica tra loro.
  • SELETTORI DI PSEUDO-CLASSI: selezionano gli elementi in base a un determinato stato.
  • SELETTORI DI PSEUDO-ELEMENTI: selezionano una parte di un elemento.
  • SELETTORI DI ATTRIBUTO: selezionano gli elementi in base ad un attributo p valore di attributo.

SELETTORI SEMPLICI

Per NOME

        p {
            text-align: center;
            color: red;
        }

per ID

esempio html:



< p id="para1"> Hello World! < /p > 

        #para1 {
            text-align: center;
            color: red;
        }


per CLASSE

esempio html:



< p class="center"> Hello World! < /p > 

        .center {
            text-align: center;
            color: red;
        }


per una CLASSE in uno specifico ELEMENTO

esempio html:



< p class="center"> Hello World! < /p > 

        p.center {
            text-align: center;
            color: red;
        }


Selettore UNIVERSALE

Con l'* va a selezionare tutti gli elementi della pagina html 



       * {

           color: red;
