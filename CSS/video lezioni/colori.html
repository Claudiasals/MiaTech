I colori possono essere assegnati in diversi modi: 1. Colori tramite nomi: color: red; backround-color: yellow; 2. Colori esadecimali (ogni colore ha un codice): color: #FFA00; /* Arancione */ background-color: #008000; /* Verde */ 3. Codici RGB (il colore è rappresentato dalla quantità di rosso, verde e blu): color: rgb(225, 0, 0); /* Rosso */ background-color: rgb(0, 0, 225) /* Blu */ 4. Codici RGBA (a sta per la trasparenza): color: rgba(225, 0. 0, 0.5); Rosso semi-trasparente */ 5. HSL e HSLA (il colore è rappresentato da: tonalità, saturazione e lumininosità): color: hsl(120, 100%, 50%); /* Verde */ background-color: hsla(0, 100%, 50%, 0.3) /* Rosso semi-trasparente */
