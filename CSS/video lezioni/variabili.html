Le variabili CSS ci consentono di memorizzare dei valori e poi riutilizzarli all'interno del nostro foglio di stile. Questo ci aiuta a mantenere il foglio pilito, organizzato e più facile da gestire. Ci consentono di apportare modifiche globali al design modificando i valori della variabile. --nome-variabile: valore; es.: 
:root {
    --colore-principale: #FF0000;
    --dimensione-testo: 18px;
}
per utilizzarla ovunque sul foglio di stile utilizzeremo la funzione "var". selettore { proprietà: var(--nome-variabile); } es.: 
 
h1 {
    color: var(--colore-principale);
    font-size: var(--dimensione-testo);
}

p {
    color: var(--colore-principale);
    font-size: calc(var(--dimensione-testo) - 2px);
}
