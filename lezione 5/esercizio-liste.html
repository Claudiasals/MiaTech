Lista non ordinata

  • il tag per creare una lista non ordinata è "ul"
  • ogni elemento dell'elenco inizia con il tag "li"
  • si possono inserire infiniti elementi in lista
  • ogni elemento della lista sarà preceduto da un punto

Lista ordinata

  1. il tag per creare una lista non ordinata è "ol"
  2. ogni elemento sarà contrassegnato da un numero in ordine crescente
  3. ogni elemento dell'elenco inizia con il tag "li"
