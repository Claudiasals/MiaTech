ESERCIZI SUL DOM

DOM (Document Object Model), ovvero la rappresentazione ad oggetti di una pagina HTML:
- Ogni elemento HTML diventa un oggetto nel DOM.
- in qst modo JavaScript può leggere, modificare o aggiungere elementi alla pagina in modo dinamico. Il document è sinonimo di DOM.
